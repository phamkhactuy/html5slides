Twitter là gì?

- Twitter là dịch vụ mạng xã hội miễn phí cho phép người sử dụng đọc, nhắn và cập nhật các mẩu tin nhỏ gọi là tweet, một dạng tiểu blog.

  • Twitter là một mạng nhắn tin.
  • Dùng Twitter để cập nhật tin tức.
  • Twitter là công cụ marketing xã hội.

Các con số ấn tượng:

Số lượng Tweet qua từng năm:

  • 5.000 tweet/ngày trong năm 2007
  • 300.000 tweet/ngày trong năm 2008
  • 2,5 triệu tweet/ngày trong năm 2009
  • 50 triệu tiếng "tuýt"/ngày trong năm 2010

API trên Twitter?


API trên Twitter hiện nay có 2 phiên bản là:

  • API v1
  • API v1.1

Các ngôn ngữ được hỗ trợ:

ActionScript/Flash, C++, Clojure, ColdFusion, Erlang, Java, JavaScript, .NET, Objective-C / Cocoa, Perl, PHP, Python, Ruby, Scala

Thư viện hỗ trợ PHP trên Twitter:

  • Oauth-php by Corollarium Technologies – an OAuth library for clients and providers
  • Services_Twitter (PEAR) by Joe Stump, David Jean Louis and Bill Shupp – a Twitter API library
  • TmhOAuth by Matt Harris – an OAuth library (supports file uploading)
  • TwitterOAuth by Abraham Williams – a Twitter API library
  • Twitter async by Jaisen Mathai – a wrapper for Twitter OAuth API (provides asynchronous calls)
